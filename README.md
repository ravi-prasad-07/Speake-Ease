# Speak-Ease 🚀

*A Web-Based Communication Application for ALS Patients*

## 📌 Overview

Speak-Ease is a web-based communication application developed to help ALS patients communicate using only **three keys**:

```text
.   -   /
```

By arranging these symbols in specific patterns, users can generate meaningful words and communicate effectively.

This project was built during our **2nd Year, 3rd Semester** as a team project focused on solving a real-world accessibility problem using technology and Data Structures.

---

## ✨ Features

* Communication using only 3 symbols
* Pattern-based word generation
* User-friendly web interface
* Fast and efficient processing
* Real-world accessibility focused solution

## 🚀 How to Run the Project

### 1️⃣ Clone the Repository

```bash
git clone <your-github-repo-link>
```

### 2️⃣ Navigate to Project Directory

```bash
cd speak-ease
```

### 3️⃣ Install Dependencies

```bash
pip install flask
```

### 4️⃣ Run the Flask Application

```bash
python app.py
```

### 5️⃣ Open in Browser

```text
http://127.0.0.1:5000
```
